# Portable desktop build

Denna mapp innehaller en separat Electron-wrapper for att bygga Odlingskalender som en portabel Windows-app.

## Kommandon

```powershell
& "C:\Program Files\nodejs\npm.cmd" install
& "C:\Program Files\nodejs\npm.cmd" run dist
```

Den portabla exe-filen hamnar i `desktop-portable/dist/`.
